# Thai-Swedish Healthcare Dictionary – Frontend

Next.js 16 + Chakra UI frontend for the Thai-Swedish Healthcare Dictionary.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Set `NEXT_PUBLIC_API_URL` in `.env.local` if the backend runs on a different URL (default: `http://localhost:5196`).

## Deployment

This app is deployed on **Render** as a Web Service. See [DEVELOPER.md](../DEVELOPER.md#deploying-on-render) for full deployment instructions.
